gpsDashboard is an app that delivers miscellaneous information gathered with the Palm Pre's GPS service. This document includes information on both gpsDashboard Free, and gpsDashboard+, if you are using the free version you may not have access to some of the features mentioned here. gpsDashboard+ will be available in the app catalog soon.

Features
  • Operates in Airplane Mode
  • Graphic satellite signal strength indicator
  • Current Speed, Direction, and Altitude readout
  • Top speed tracking. Can be reset by tapping on the current information box, or by shaking the device. Shaking reset can be configured for any other reset from within preferences.
  • All-time record tracking with locations. This may be accessed by tapping on the title bar.
  • Colored speed indicates whether you are traveling faster or slower than average speed. This can be turned off in preferences
  • Trip Information Tracking. All values can be reset by tapping on the trip information box.
    • Average Speed
    • Distance Traveled
    • Distance traveled from the inital point. The initial point is recorded when the app is first loaded, and the minium error tolerance is satisfied.
    • Lifetime Distance
  • A reverse location feature that will display your current address.

Preferences Explained

On Startup - Determines the trip reset behavior when the program is loaded.

Units - Allows the user to select metric or imperial units of measure.

Backlight - Allows the user to force the screen to stay lit while the program is running.

Max Error - Maximum allowed error in the location fixes.

When Shook - Allows the user to select what gets reset (if anything) when the device is shook.

Colored Speed - Allows the user to turn off the colored speed feature.

Average Speed - Determines whether the total distance traveled, or merely the distance traveled from the inital location is used for the average speed calculation.

Frequently Asked Questions

I get an error saying I haven't accepted the Terms of Service

    Go to Location Services (Third page in the launcher), and goto the preferences menu. From there tap "Locate Me Using" and tap "Google Services". It will then ask you to accept the terms of service.
I get an error saying to ensure GPS fixes are turned on
    Go to Location Services (Third page in the launcher), and goto the preferences menu. From there tap "Locate Me Using" and tap "GPS".
I get an error saying I'm blacklisted
    The only time I've seen this happen is when the app asks your permission to use Location Services and you either decline, or close the app without responding. As far as I know, the only way to clear the error is to restart the phone.
Top Speed goes down unexpectedly
    By default the app will reset the top speed when you shake the phone. This behavior can be changed in preferences.
Whats the deal with the accuracy
    By default trip information will not be tracked unless the accuracy is correct within ten meters. This can be changed in preferences.
All-Time records don't seem to be updating
    All-Time records are hard coded to require a minimum of 10 meter accuracy. If the signal is not strong enough, these records will not be updated.
The 'Get Address' button doesn't work
    This is an issue with the reverse location service in webos that I have no control over. Unfortunately the only way I've found to fix it is to restart the phone.
